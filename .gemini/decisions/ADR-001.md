# ADR-001: AI Workflow Roles

Context:

- This repo is a template to run an AI-driven PR workflow.

Decision:

- Source of truth is .flow/\*
- Claude writes .flow/plan.md and .flow/task.md
- Codex executes .flow/task.md and opens PRs
- Claude reviews PRs
- Gemini provides decision/constraints only, stored as ADRs

Status: Accepted
